# Autonomous Booking Agent Plan

## Purpose

Turn Sonic-Forage/Jimsky/DJ/community assets into a safe booking pipeline without autonomous spam.

## Agent lanes

1. **Research lane:** collect public venue/festival/community info.
2. **Fit lane:** classify whether the opportunity matches Sonic-Forage values.
3. **Asset lane:** attach approved one-sheet, repo links, Pages deck, promo assets.
4. **Draft lane:** write a human-reviewed pitch.
5. **Approval lane:** require explicit approval before sending.
6. **Follow-up lane:** prepare reminders, never send without approval.

## Never autonomous

- sending email/DMs
- signing contracts
- negotiating money
- public posting
- claiming availability/fees
- claiming medical/harm-reduction training
- scraping private info

## Best first demo

Use a fictional venue lead payload, generate a one-sheet draft, run verifier, and show the approval ledger.
